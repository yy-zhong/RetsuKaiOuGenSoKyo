# 《烈海王似乎打算在幻想乡挑战强者们的样子》懒人包

本仓库存档了贴吧用户[王子 2326](https://tieba.baidu.com/home/main?id=tb.1.e5ba04dc.L15sxyYylj7xSZqTiL_BoQ)在刃牙吧刊载的安科贴[《【安科】烈海王似乎打算在幻想乡挑战强者们的样子》](https://tieba.baidu.com/p/6710222599)和[《【安科】烈海王似乎打算在幻想乡挑战强者们的样子 第二季》](https://tieba.baidu.com/p/6851377332)。出于补番方便，只收录了原作者的所有帖子，没有评论和其他用户的发言。

依据贴吧的“只看楼主”的功能，对页面进行划分章节。章节的分割不一定代表剧情的分割。

每个章节下由多个图片文件组成，即原作者的贴子截图。

举例：`S01E01`文件夹下的文件即《【安科】烈海王似乎打算在幻想乡挑战强者们的样子》贴的第一页；`S02E15`文件夹下的文件即《【安科】烈海王似乎打算在幻想乡挑战强者们的样子 第二季》贴的第十五页。

**所有的图片文件均在 Chapters 文件夹下**

如果不想阅读单个的图片，本仓库也提供合并后的PDF阅读（每个章节一个文件），可以前往release页面查看。
